# luke-portfolio

Personal portfolio site for Luke Courtright. Plain static HTML/CSS/JS, no build step, using the "Neon Night" brand system (cyan/purple/gold on midnight navy, light + dark mode).

## Run locally

```bash
python -m http.server
```

Then open `http://localhost:8000`.

## Deploy

Hosted on GitHub Pages, deploying from the `main` branch root.
